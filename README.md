## 프로젝트 소개
  - 여러 가지 AI model 개발할 때 각 model의 개발 일정과 현재 상태 등 일정을 관리하는 프로그램입니다.
  - 각 task의 완료율, due date, 책임자(개발자), 현재 model의 Accuracy 정보를 기록하고 확인할 수 있습니다.

## 포함 기능
  - CRUD 기능
  - DueDate 검색 기능
  - 책임자(Engineer) 검색 기능
  - Task(model) 검색 기능
  - 파일 저장 / 불러오기

## 개발 환경 및 언어
  - git
  - vscode
  - C 언어
